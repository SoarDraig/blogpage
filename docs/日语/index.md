# 日本語

日本語は日本で話されている言語で、独特の文法や文字体系があります。このセクションでは、日本語の基本的な文法、語彙、発音について学びます。

## 主な特徴
- **ひらがな・カタカナ・漢字**：日本語は3つの文字を使用します。
- **敬語**：敬語が重要な文化的要素で、相手に対する敬意を表します。
- **文法**：日本語の文法は、主語-目的語-動詞の順序です。
