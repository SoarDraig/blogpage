# Unity 引擎

Unity 是一个跨平台的游戏引擎，广泛应用于2D和3D游戏开发。它提供了一整套开发工具和资源，支持图形渲染、物理模拟和动画制作，成为独立开发者和大型工作室的首选。

## 主要特点
- **实时渲染**：支持高效的实时图形渲染和物理引擎。
- **多平台发布**：支持Windows、Mac、iOS、Android等多个平台的发布。
- **强大的社区**：拥有活跃的开发者社区和丰富的插件生态系统。
